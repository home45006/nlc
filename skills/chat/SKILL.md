# 智能问答

通用问答和闲聊功能，包括自由聊天、车辆问答和天气查询。

## 能力描述

### free_chat - 自由聊天

日常对话和问候，处理通用闲聊内容。

**参数：**

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| message | string | 否 | 用户消息内容 |

**示例：**

- 你好 -> `{ message: "你好" }`
- 今天天气怎么样 -> `{ message: "今天天气怎么样" }`
- 讲个笑话 -> `{ message: "讲个笑话" }`
- 谢谢你 -> `{ message: "谢谢你" }`
- 再见 -> `{ message: "再见" }`

**支持的对话类型：**

- 问候语：你好、嗨、早上好等
- 感谢语：谢谢、多谢等
- 告别语：再见、拜拜等
- 日常闲聊：讲笑话、聊天等

---

### vehicle_qa - 车辆问答

回答关于车辆功能的问题。

**参数：**

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| question | string | 是 | 用户的问题 |

**示例：**

- 这车油耗多少 -> `{ question: "这车油耗多少" }`
- 怎么打开自动泊车 -> `{ question: "怎么打开自动泊车" }`
- 续航里程是多少 -> `{ question: "续航里程是多少" }`
- 油箱多大 -> `{ question: "油箱多大" }`
- 怎么连接蓝牙 -> `{ question: "怎么连接蓝牙" }`

**常见问题类型：**

- 油耗和续航
- 功能使用说明
- 车辆参数
- 系统设置

---

### weather_query - 天气查询

查询指定城市和日期的天气情况。

**参数：**

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| city | string | 否 | 城市名称（默认当前位置） |
| date | string | 否 | 日期：今天、明天、后天等 |

**示例：**

- 今天天气怎么样 -> `{ date: "今天" }`
- 明天会下雨吗 -> `{ date: "明天" }`
- 北京天气如何 -> `{ city: "北京" }`
- 上海明天天气 -> `{ city: "上海", date: "明天" }`

## 触发关键词

- 你好、谢谢、再见
- 油耗、续航、功能
- 天气、下雨、温度

## 优先级说明

智能问答的优先级较低（priority: 10），当用户输入匹配其他特定领域时，会优先路由到对应 Skill。只有当输入不匹配任何特定领域时，才会由智能问答处理。

## 使用场景

1. **日常问候**: 用户说"你好"、"早上好"等
2. **车辆知识**: 用户询问车辆相关的问题
3. **天气查询**: 用户想知道天气情况
4. **闲聊娱乐**: 用户想要聊天或听笑话

## 注意事项

1. 车辆问答基于预设知识库，复杂问题建议查阅手册
2. 天气查询需要联网获取实时数据
3. 自由聊天模式会保持对话上下文
