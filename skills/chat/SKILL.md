# 智能问答

实用工具类能力，包括车辆问答、天气查询、计算器和系统信息查询。

## 能力描述

### vehicle_qa - 车辆问答

回答关于车辆功能的问题。

**参数：**

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| question | string | 是 | 用户的问题 |

**示例：**

- 这车油耗多少 -> `{ question: "这车油耗多少" }`
- 怎么打开自动泊车 -> `{ question: "怎么打开自动泊车" }`
- 续航里程是多少 -> `{ question: "续航里程是多少" }`
- 油箱多大 -> `{ question: "油箱多大" }`
- 怎么连接蓝牙 -> `{ question: "怎么连接蓝牙" }`

**常见问题类型：**

- 油耗和续航
- 功能使用说明
- 车辆参数
- 系统设置

---

### weather_query - 天气查询

查询指定城市和日期的天气情况。

**参数：**

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| city | string | 否 | 城市名称（默认当前位置） |
| date | string | 否 | 日期：今天、明天、后天等 |

**示例：**

- 今天天气怎么样 -> `{ date: "今天" }`
- 明天会下雨吗 -> `{ date: "明天" }`
- 北京天气如何 -> `{ city: "北京" }`
- 上海明天天气 -> `{ city: "上海", date: "明天" }`

---

### calculator - 计算器

执行数学计算。

**参数：**

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| expression | string | 是 | 数学表达式 |

**示例：**

- 计算 2 加 3 -> `{ expression: "2+3" }`
- 100 除以 5 等于多少 -> `{ expression: "100/5" }`
- 算一下 123 乘以 456 -> `{ expression: "123*456" }`

---

### system_info - 系统信息

查询系统状态信息。

**参数：**

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| infoType | string | 否 | 信息类型（cpu/memory/disk/network/all） |

**示例：**

- 系统状态怎么样 -> `{ infoType: "all" }`
- CPU 使用率多少 -> `{ infoType: "cpu" }`
- 内存还剩多少 -> `{ infoType: "memory" }`

---

## 触发关键词

- 油耗、续航、功能
- 天气、下雨、温度
- 计算、等于、加减乘除
- 系统、CPU、内存

## 优先级说明

智能问答的优先级较低（priority: 10），当用户输入匹配其他特定领域时，会优先路由到对应 Skill。

## 使用场景

1. **车辆知识**: 用户询问车辆相关的问题
2. **天气查询**: 用户想知道天气情况
3. **数学计算**: 用户需要进行简单计算
4. **系统状态**: 用户想了解系统运行状态

## 注意事项

1. 车辆问答基于预设知识库，复杂问题建议查阅手册
2. 天气查询需要联网获取实时数据
3. 计算器支持基本四则运算
